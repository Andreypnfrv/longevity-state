# Индикаторы для шкалы Public trust (science & medicine)

**Статус:** черновик плана, к выполнению вернёмся позже.

**Открытый ход:** если одного балла недостаточно, логично **бить шкалу**: отдельные субсчёты или отдельные `SocietalReadinessClaim` по столпам A/B/C (или «институции vs новизна»), с явной агрегацией в продукте. Ниже — база для одной шкалы с правилом min-pillar; при сплите столпы превращаются в самостоятельные оси.

## Принцип агрегации

- **Один общий балл** = уровень по **самому слабому** из трёх столпов, релевантных для longevity (новые терапии, клинические исследования, национальные программы данных/скрининга) — зафиксировано в комментарии к `SOCIETAL_READINESS_CLAIM_SCALES` в [src/schema/index.ts](src/schema/index.ts).
- Для каждой страны в `links` желательно явно помечать, **какой столп** тянет оценку вниз (в `comment`), если это не очевидно из текста.

## Три столпа (что именно мерить)


| Столп                                         | Смысл                                                                   | Типичные источники вопросов                                                                                   |
| --------------------------------------------- | ----------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------- |
| **A. Клиницисты / система**                   | Доверие к врачам, больницам, «медицине в целом»                         | Eurobarometer (health), national health barometers, Ipsos Veracity-style institution rankings (UK NHS и т.д.) |
| **B. Наука / исследования**                   | Доверие к учёным, исследованиям, университетам в контексте здоровья     | Wellcome Global Monitor, national science attitude surveys                                                    |
| **C. Государственные органы здравоохранения** | Доверие к минздраву, CDC-аналогу, регулятору как источнику рекомендаций | Eurobarometer, Edelman Trust (осторожно: government ≠ минздрав), национальные волны                           |


**Не смешивать** с QUACKERY_RESISTANCE в [src/schema/index.ts](src/schema/index.ts): высокое доверие к «официальной медицине» сосуществует с большим рынком БАД; это отдельная ось.

## Формулировки индикаторов (семейства вопросов)

Не пороги в %, а **карта покрытия**: какие темы должны быть закрыты доказательством для присвоения рунга.

1. **Базовое доверие (A/B)** — врачи / больницы / учёные; для 3+ желательно две волны или золотой источник + согласующийся национальный.
2. **Институциональное здравоохранение (C)** — национальный орган политики / эпиднадзора; критично для данных и мандатов.
3. **Новизна и коллективные риски (2 vs 3)** — генная/клеточная терапия, новые технологии в медицине, участие в исследованиях / данные для науки; низко при хорошем A → типичный сигнал рунга 2.
4. **Поляризация** — разрыв по партиям/регионам; квалификатор в narrative, не обязательно отдельный индекс в UI.
5. **Поведенческие прокси** — вакцинация/скрининг только с контролем доступа; поддержка, не замена опросов.

## Привязка к рунгам (качественно)

- **0**: кризис легитимности, вето на консенсусные меры.
- **1**: устойчиво ниже пира по A или B или C; хроническая неохота к базовой профилактике при нормальном доступе.
- **2**: A ок, но C или новизна/данные слабые или сильная поляризация без полного блока.
- **3**: большинства по A и B; C не провален; новизна не табу для политического обсуждения.
- **4–5**: высокие уровни на стандартных шкалах; 5 — устойчивость через основные расколы + межстрановой бенчмарк.
